# Braun Design Co

Boutique Shopify Plus agency in Austin, Texas. We build custom storefronts for ecommerce brands and help them migrate, perform, and stay discoverable as search moves toward AI.

Founded in 2016, we work as a single point of contact for a small roster of mid-market brands, writing code that fits your theme architecture instead of fighting it.

## What we do

- **Custom theme development.** Built on native Shopify sections, not page builders, with schema we write ourselves so your content stays under your control.
- **Platform migrations.** Magento and WooCommerce to Shopify Plus, with attention to SEO, redirects, and data integrity.
- **Conversion and performance.** Front-end work focused on speed, clean markup, and measurable lift.
- **AI Search Optimization (AEO / AISO).** Structuring catalogs and content so brands stay visible as shoppers and AI agents shift to AI-driven search, including Agentic Storefront configuration.

## How we build

We work close to the platform: Liquid, GraphQL, custom web pixels, and Shopify Flow, integrated into your existing theme rather than bolted on top. Clean, semantically correct code is the default, not an upgrade.
